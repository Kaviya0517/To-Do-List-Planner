
# Multi-Card To-Do Planner

A dynamic **task management web application** built using **HTML, CSS, and JavaScript** that allows users to create multiple task cards, organize tasks by date, and manage them interactively.

This project started as a simple to-do list but gradually evolved into a **multi-card planner** where each card represents a separate task group for a specific date.

## 🚀 Features
- ➕ Create new task cards using a floating plus button
- 📅 Assign tasks to a specific date
- ✅ Mark tasks as completed
- ✏ Edit tasks anytime
- ❌ Delete individual tasks
- 🗂 Save task cards to the board
- 🗑 Delete entire cards
- 📱 Dynamic UI created using JavaScript DOM manipulation

## 🛠 Technologies Used

- **HTML5** – Structure of the application  
- **CSS3** – Styling and layout  
- JavaScript – Dynamic functionality and DOM manipulation

## ⚡ Challenges Faced

One of the biggest challenges was handling **dynamically created elements**.  
Each new card required its own tasks, buttons, and event handling logic.

Another challenge was managing the UI behavior such as:

- Opening new cards in the center of the screen
- Moving cards into the layout after saving
- Keeping multiple cards properly aligned

Solving these problems helped strengthen my understanding of **JavaScript DOM operations and event handling**.
